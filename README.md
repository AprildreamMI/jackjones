## 杰克琼斯

### 一：项目描述

​	一个前端简单项目，仿作杰克琼斯官网，有三个页面

```
	页面1：主页 （轮播图）
	
	页面2：商品详情页 （图片放大镜）
	
	页面1：购物车页 （上下滑显示）
```
